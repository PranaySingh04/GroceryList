# GroceryList
## Features
* Use this app to keep track of your groceries!
* Enter your grocery list items into the text box.
* Use the Clear button at the bottom of the screen to delete your list once you're done!
### Future Features
* Making your list persistent, so your list is saved if you close out of the app and reopen it later. 
## Creating this app
* This was the first app I created completely from scratch, without any prior knowledge of apps like this, or the help of tutorials.
